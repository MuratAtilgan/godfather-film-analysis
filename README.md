## Explore The Website

This project contains **9 comprehensive pages** covering:

- **Home** - Hero section with iconic quotes
- **Introduction** - Purpose and target audience
- **Design** - Design choices and philosophy
- **Films** - Complete trilogy analysis with statistics
- **Characters** - In-depth character profiles
- **Media** - Featured content and visuals
- **Info** - Technical documentation
- **Validation** - W3C compliance screenshots
- **Contact** - Interactive feedback form

**Visit the website to explore all pages in detail!**
